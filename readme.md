# ValidateMyGrid

A set of validation rules about power in OSM.

## Add to JOSM

To add the ruleset to JOSM:

* Edit -> Preferences
* Data Validator
* Tag Checker Rules (2nd tab)
* search "Power QA", select the item and add it with the ">" button in the middle of the screen

## Use it in Osmose

Some of these validation rules are also available on Osmose (item 7040):

* [list](https://osmose.openstreetmap.fr/en/issues/open?item=7040) (be patient, it takes some time for the page to load the full list of issues)
* [map](https://osmose.openstreetmap.fr/en/map/#item=7040&level=1%2C2%2C3)
* [progress](https://osmose.openstreetmap.fr/en/issues/graph.png?item=7040)

## Documentation

Please read more about the different validation rules on the [OpenStreetMap Quality Assurance page](https://wiki.openstreetmap.org/wiki/Power_networks/Quality_Assurance).
